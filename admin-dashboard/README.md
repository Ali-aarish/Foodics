# Admin Dashboard 🖥️

The web dashboard that **restaurant owners and managers** use to run their
business.

## What admins can do here
- Manage the menu and prices
- Track inventory and stock
- Manage staff and roles
- View sales and business analytics
- Set up offers and loyalty programs
- Manage multiple branches/markets

## Folders
```
admin-dashboard/
├── pages/        → Dashboard pages (Menu, Inventory, Staff, Analytics, Branches)
├── components/   → Reusable UI pieces (tables, charts, forms)
├── services/     → Code that talks to the backend (API calls)
└── assets/       → Images, icons, styles
```

## Suggested Tech
React (web)
