# POS App 🧾

The **Point of Sale** app that restaurant staff (cashiers & waiters) use,
usually on a tablet.

## What staff can do here
- Take customer orders
- Manage tables and seating
- Split or merge bills
- Accept payments (card, cash, digital)
- Send orders to the kitchen
- Print or share receipts

## Folders
```
pos-app/
├── screens/      → App pages (Order, Tables, Checkout, Payment, Receipt)
├── components/   → Reusable UI pieces (menu items, cart, keypad)
├── services/     → Code that talks to the backend (API calls)
└── assets/       → Images, icons, fonts
```

## Suggested Tech
React Native / Flutter (tablet-friendly)
