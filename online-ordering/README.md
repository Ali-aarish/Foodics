# Online Ordering 🛒

The site/app that **customers** use to order food online for pickup or delivery.

## What customers can do here
- Browse the restaurant menu
- Add items to a cart
- Choose pickup or delivery
- Pay online securely
- Track their order
- Save favorites and reorder

## Folders
```
online-ordering/
├── pages/        → Site pages (Menu, Cart, Checkout, Order Tracking)
├── components/   → Reusable UI pieces (menu cards, cart, item options)
├── services/     → Code that talks to the backend (API calls)
└── assets/       → Images, icons, styles
```

## Suggested Tech
React / Next.js (web)
